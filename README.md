# 📄 НИР-Форматтер — ГОСТ 7.32-2017

Streamlit-приложение для автоматического оформления отчёта о НИР строго по ГОСТ 7.32-2017.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
cd gost_formatter
streamlit run app.py
```

## Структура проекта

```
gost_formatter/
├── app.py              # Streamlit UI
├── parser.py           # Парсер входного текста
├── docx_builder.py     # Генератор DOCX (python-docx)
├── styles/
│   └── gost_styles.py  # Константы ГОСТ 7.32-2017
└── requirements.txt
```

## Синтаксис разметки

| Элемент            | Синтаксис                                          |
|--------------------|----------------------------------------------------|
| Раздел             | `# 1 НАЗВАНИЕ РАЗДЕЛА`                             |
| Подраздел          | `## 1.1 Название подраздела`                       |
| Пункт              | `### 1.1.1 Название пункта`                        |
| Рисунок            | `[рисунок: файл.png | Подпись к рисунку]`          |
| Таблица            | `[таблица]` ... `[/таблица | Название таблицы]`    |
| Маркированный список | `- Элемент`                                      |
| Нумерованный список | `1) Первый пункт`                                 |
| Разрыв страницы    | `---`                                              |

## Параметры ГОСТ 7.32-2017

### Страница (А4)
- Поле левое: **30 мм**
- Поле правое: **10 мм**
- Поле верхнее: **20 мм**
- Поле нижнее: **20 мм**

### Основной текст
- Шрифт: **Times New Roman 14 пт**
- Межстрочный интервал: **полуторный**
- Абзацный отступ: **1,25 см**
- Выравнивание: **по ширине**

### Заголовки
- H1 (разделы): 14 пт, жирный, ПРОПИСНЫЕ, с новой страницы, по центру
- H2 (подразделы): 14 пт, жирный, с абзацного отступа
- H3 (пункты): 14 пт, жирный курсив

### Колонтитулы
- Нижний: номер страницы по центру, 12 пт
- Первая страница (титул) — без номера

### Подписи
- Рисунки: «Рисунок N — Название» под рисунком, по центру
- Таблицы: «Таблица N — Название» над таблицей, слева
