# 📄 Форматирование НИР по ГОСТ 7.32-2017

Автоматическое оформление отчёта о научно-исследовательской работе в соответствии с ГОСТ 7.32-2017.

## ✨ Возможности

### Два режима работы:

1. **Текст с Markdown-разметкой** — ввод текста со специальными тегами
2. **Загрузка DOCX файла** — автоматическое извлечение структуры из неформатированного Word-документа

### Поддерживаемые элементы:

- ✅ Титульный лист по ГОСТ
- ✅ Реферат (обязательный элемент)
- ✅ Автоматическое содержание (TOC)
- ✅ Разделы и подразделы с нумерацией
- ✅ Таблицы с подписями
- ✅ Рисунки с подписями (извлекаются из DOCX или загружаются отдельно)
- ✅ Списки (маркированные и нумерованные)
- ✅ Формулы
- ✅ Нижние колонтитулы с номерами страниц

## 🚀 Запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
streamlit run app.py
```

## 📥 Режим 1: Загрузка DOCX файла

1. Выберите режим **"Загрузить DOCX файл"**
2. Загрузите неформатированный Word-документ
3. Система автоматически:
   - Извлечёт текст из всех параграфов
   - Распознает заголовки по стилям Word (Heading 1, 2, 3)
   - Найдет таблицы и изображения
   - Определит специальные разделы (ВВЕДЕНИЕ, ЗАКЛЮЧЕНИЕ и т.д.)
4. Проверьте распознанную структуру
5. Заполните метаданные (министерство, организация, название НИР и т.д.)
6. Нажмите **"Сформировать DOCX"**

### Советы для лучшего распознавания:

- Используйте стили заголовков в Word (Heading 1, Heading 2, etc.)
- Специальные разделы (**ВВЕДЕНИЕ**, **ЗАКЛЮЧЕНИЕ**, **РЕФЕРАТ**, **СОДЕРЖАНИЕ**) должны быть написаны ЗАГЛАВНЫМИ буквами
- Таблицы должны быть оформлены как таблицы Word (не текстом!)

## 📝 Режим 2: Текст с разметкой

Используйте следующий синтаксис:

| Элемент | Синтаксис |
|---|---|
| Раздел (H1) | `# 1 НАЗВАНИЕ РАЗДЕЛА` |
| Подраздел (H2) | `## 1.1 Название подраздела` |
| Пункт (H3) | `### 1.1.1 Название пункта` |
| Рисунок | `[рисунок: имя_файла.png \| Подпись]` |
| Таблица | `[таблица]` ... `[/таблица \| Название]` |
| Маркированный список | `- Элемент списка` |
| Нумерованный список | `1) Первый пункт` |
| Разрыв страницы | `---` |

## 📏 Параметры оформления по ГОСТ 7.32-2017

### Страница (А4)
- Поле левое: 30 мм
- Поле правое: 10 мм
- Поле верхнее: 20 мм
- Поле нижнее: 20 мм

### Текст
- Шрифт: Times New Roman 14 пт
- Межстрочный интервал: полуторный
- Отступ абзаца: 1,25 см
- Выравнивание: по ширине

### Заголовки
- H1: 14 пт, жирный, ПРОПИСНЫЕ, с новой страницы
- H2: 14 пт, жирный, с отступом
- H3: 14 пт, жирный курсив
- Нижний колонтитул: № стр. по центру, 12 пт

## ⚙️ Метаданные

При формировании отчёта заполняются:
- Министерство / ведомство
- Организация
- Название НИР
- Шифр темы
- УДК
- Инвентарный номер
- Город и год
- Руководитель и исполнители
